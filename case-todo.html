Case Study · 01

智能待办清单

市面上的待办软件要么太复杂——Notion 级的学习成本，要么太简陋——只能打勾。做一个"刚好够用但又好看"的。

问题

选择困难

待办软件两极分化：一端是 Notion / TickTick 功能爆炸，另一端是系统备忘录啥也没有。中间地带——够用、好看、打开就用的——不存在。

解法

刚好够用

不做大而全。五套主题让你挑喜欢的，番茄钟内嵌不跳转，子任务拆解不臃肿。PWA 装手机上像原生 App，不用下载。

五套主题

Cashmere / Noir / Sage / Mauve / Sepia，一键切换，不用调色

番茄钟

内嵌在任务里，不跳页面。开始计时、完成休息，专注心流

子任务拆解

大任务拆小步，一步步勾，不压脑

拖拽排序

手指拖拽调优先级，比打字改序号快十倍

离线 PWA

没网也能用。Service Worker 缓存，打开即用

导入导出

JSON 格式，数据永远在你手里，不锁

PWAService WorkerIndexedDBVanilla JS零依赖响应式

"客户不关心你用 React 还是 Vue。客户关心：这东西装我手机上能用吗？好看吗？会丢数据吗？"

—— 做完这个项目才懂的道理
